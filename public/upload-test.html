MultiImageUpload Component Test

Test Steps:

  1. ไปที่ CMS: http://localhost:5173/products
  2. กดปุ่ม "Add Product"
  3. เลือก Tab "Images"
  4. ดูว่ามี MultiImageUpload แสดงไหม
  5. เช็ค Console สำหรับ debug logs

Expected Behavior:

  • ควรเห็น heading "Product Images"
  • ควรเห็น debug info แสดง formData.images
  • ควรเห็น "No images added yet" card
  • ควรเห็น "Add New Image" upload area

Troubleshooting:

  • ถ้าไม่เห็น Tab "Images" → เช็ค TabsList rendering
  • ถ้าไม่เห็น MultiImageUpload → เช็ค import/export
  • ถ้าไม่เห็น ImageUpload → เช็ค authentication
  • ถ้า upload ไม่ได้ → เช็ค backend API

Console Commands to Test:
